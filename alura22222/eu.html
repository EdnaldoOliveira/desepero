Principais Comandos para Criar um Site

Comando HTML Básico

O comando <html> é o ponto de partida para qualquer documento HTML. Este comando define que o conteúdo dentro dele é um documento HTML.

Comando CSS Básico

O comando style é usado para definir o estilo de um documento HTML. Pode ser incluído diretamente na página HTML ou referenciado através de um arquivo externo.
